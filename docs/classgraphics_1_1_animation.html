Detailed Description

This class allows to perform animations based on multiple frames.

Author
Kevin Fedyna
Date
16/11/2023

It can be called inside a try catch block or using the helper function Animation.load but it won't raise an exception.

Warning
The frames must be in PNG format and be named using framesName<X>.png with X starting at 1.

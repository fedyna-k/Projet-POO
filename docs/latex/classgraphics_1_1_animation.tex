\doxysubsection{Detailed Description}
This class allows to perform animations based on multiple frames. 

\begin{DoxyAuthor}{Author}
Kevin Fedyna 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
16/11/2023
\end{DoxyDate}


It can be called inside a {\ttfamily try catch} block or using the helper function Animation.\+load but it won\textquotesingle{}t raise an exception.

\begin{DoxyWarning}{Warning}
The frames must be in {\bfseries{PNG}} format and be named using {\ttfamily frames\+Name\texorpdfstring{$<$}{<}X\texorpdfstring{$>$}{>}.png} with X starting at 1. 
\end{DoxyWarning}

Add key to the Key\+Stack inputs. 

The key is in the format of what Key\+Stroke can read.

Whenever the given key is pressed, the Key\+Stack will raise a {\ttfamily Add\+Key\+Action} event. Whenever the given key is released, the Key\+Stack will raise a {\ttfamily Remove\+Key\+Action} event.

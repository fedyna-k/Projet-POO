◆ listenTo()

void graphics.KeyStack.listenTo ( String  key)

Add key to the KeyStack inputs.

The key is in the format of what KeyStroke can read.

Whenever the given key is pressed, the KeyStack will raise a AddKeyAction event. Whenever the given key is released, the KeyStack will raise a RemoveKeyAction event.
